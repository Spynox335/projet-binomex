1. À quoi sert la commande git init ?

la commande "git init" sert a initier un dossier que l'on vient de créer

2. Quelle est la différence entre git add et git commit ?

git add ajoute un fichier tandis que git commit permet de sauvegarder les modifications

3. À quoi sert la commande git status ?

git status sert a voir le nombre de commit fait sur le document et voir ce que chaque commit modifie


4. Quelle commande permet d’afficher la liste des commit déjà réalisés ?

git status

5. Quelle est la différence entre git commit et git push ?

git commit permet de sauvegarder en local alors que git push sauvegarde sur github

6. Que se passe-t-il quand on fusionne une branche sur GitHub ?

cela créer une branche sur github et sur notre doc

7. Quelle commande permet de créer une nouvelle branche ? Et quelle commande permet d’y aller ?

git checkout et git branche

8. Quelle commande utilises-tu pour récupérer un projet GitHub sur ton ordinateur ?

git pull

9. Que contient un commit ?

un commit contient les modification fait par l’utilisateur avant la sauvegarde ( commit )

10. Quelle est la différence entre Git et GitHub ?

git et la plateforme permettant de faire le code tandis que github permet de les stocker en ligne
